# Data Source

**Dataset:** IBM Telco Customer Churn

**Source:** Kaggle

**URL:** https://kaggle.com/datasets/blastchar/telco-customer-churn

**Licence:** CC0 — Public Domain

**File:** WA_Fn-UseC_-Telco-Customer-Churn.csv (~1MB)

**Rows:** 7,043 customers

**Columns:** 21

## To Reproduce This Project

1. Download WA_Fn-UseC_-Telco-Customer-Churn.csv from the URL above
2. Place it in this /data folder
3. Open CustomerChurnAnalytics.pbix in Power BI Desktop
4. If prompted, update the data source path to your local CSV file
5. Click Refresh — all queries will rebuild automatically
