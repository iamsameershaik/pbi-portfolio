# Data Source

**Dataset:** UCI Online Retail II
**Source:** UC Irvine Machine Learning Repository
**URL:** https://archive.ics.uci.edu/dataset/502/online+retail+ii
**Licence:** CC BY 4.0 — free to share and adapt with attribution
**File:** online_retail_II.xlsx (~45MB — not committed due to file size)

## To Reproduce This Project

1. Download online_retail_II.xlsx from the URL above
2. Place it in this /data folder
3. Open SalesCommandCenter.pbix in Power BI Desktop
4. If prompted, update the data source path to point to your
   local copy of online_retail_II.xlsx
5. Click Refresh — all queries will rebuild automatically
